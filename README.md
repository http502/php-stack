# Docker PHP Stack

All in one php fpm image, Supported frameworks:

 - Laravel
 - Symfony
 - Slim
 - Codeigniter
 - YII
 - Etc

## Available versions:

- 5.6-fpm
- 7.1-fpm
- 7.2-fpm
- 7.3-fpm
- 7.4-fpm
